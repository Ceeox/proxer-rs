Liefert eine Liste der aktuellen Konferenzen.

Fields

id: u64

Die Konferenz-ID

topic: String

Der Titel einer Konferenz. Dies ist entweder der Benutzername des Gesprächspartners, oder das Thema einer Gruppenkonferenz.

topic_custom: String

Der Titel, der vom Benutzer festgelegt wurde.

count: u64

Anzahl der Konferenzteilnehmer.

group: bool

true oder false. true Falls es sich um eine Gruppenkonferenz handelt. Ansonsten false.

timestamp_end: String

Zeitstempel der letzten Nachricht.

read: bool

true oder false: true falls Benutzer letzte Nachricht dieser Konferenz gelesen hat.

read_count: u64

Anzahl der ungelesenen Nachrichten.

read_mid: u64

Die Message-ID der letzten gelesenen Nachricht.

image: String

Bild der Konferenz. Format: "Bild-Typ:Bild-ID". Typ ist meistens "avatar" und ID beispielsweise "62_yF5zd7.jpg". Avatare haben den folgenden Link: http://cdn.proxer.me/avatar/tn/[Bild-ID]
